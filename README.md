# Project Tournament Results

## How to run the application

* Save file in the root folder
* Follow the command structure :
```sh
    $ psql                          -> open psql
    $ \i tournament.sql             -> create database and tables
    $ \q                            -> exit psql
    $ python tournament_test.py     -> test tournament.py
```
